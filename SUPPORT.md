# Support

## Where to Ask Questions

Use GitHub Discussions for:

- General usage questions.
- Architecture and design discussions.
- Integration help.
- Open-ended conversations that do not represent a confirmed bug or concrete feature request.

Use GitHub Issues for:

- Bug reports.
- Feature requests.
- Documentation improvements.

When opening a discussion or issue, include enough context to reproduce or understand the request.

## What to Include

For storage-related questions:

- Provider package in use: `s3`, `azure`, `cloudflare`, or `kysely`.
- Runtime: Node.js, Cloudflare Workers, or another environment.
- Relevant provider details: S3-compatible backend, Azure Blob, Cosmos DB, D1, PostgreSQL, etc.
- Minimal code sample or reproduction steps.

## Security Issues

Do not use public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md).
